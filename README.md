# ДЗ #2 - Робота зі змінними, операторами та рядками

## Виконані завдання

### 1. Операції з інкрементом/декрементом
- Реалізована робота з префіксним та постфіксним інкрементом/декрементом
- Використана змінна `i` для демонстрації операцій `++i`, `i++`, `--i`, `i--`

### 2. Оператори присвоєння
- Створена змінна `myTest` зі значенням 20
- Застосовані оператори присвоєння: `+=`, `-=`, `*=`, `/=`, `%=`
- Продемонстровані різні математичні операції з присвоєнням

### 3. Робота з об'єктом Math
- Використані властивості та методи об'єкта Math:
  - Отримано значення константи Pi (`Math.PI`)
  - Виконано округлення числа 89.279 (`Math.round`)
  - Згенеровано випадкове число від 0 до 10 (`Math.random`)
  - Обчислено число 3 у степені 5 (`Math.pow`)

### 4. Робота з об'єктами та рядками
- Створено об'єкт `strObj` з текстовим вмістом
- Реалізовано отримання довжини рядка
- Базовий рядок: "Мама мила раму, рама мила маму"

### 5. Пошук підрядка
- Реалізовано пошук тексту "рама" в рядку
- Отримано індекс входження (`isRamaPos`)
- Створено булеве значення наявності тексту (`isRama`)

### 6. Заміна підрядків
- Виконана заміна частин тексту:
  - "мила" → "миє"
  - "рама" → "Рама"
  - "мила" → "тримає"
- Результат: "Мама миє раму, Рама тримає маму"

### 7. Перетворення регістру
- Робота з рядком "some STRING"
- Реалізовано перетворення у верхній регістр (`toUpperCase`)
- Реалізовано перетворення у нижній регістр (`toLowerCase`)

## Запуск проекту

1. Встановлення залежностей:
```bash
npm install
# або
yarn
```

2. Запуск тестів:
```bash
npm test
# або
yarn test
```
